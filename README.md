#  Attendance Management System

A streamlined, web-based solution for managing employee attendance, tracking records, and administrative oversight. Built with a focus on simplicity and ease of use.

---

## Features

* **Secure Authentication**: Multi-level login system for Admins and Employees.
* **Employee Management**: Admins can easily add, view, and "discharge" employees from a centralized dashboard.
* **Attendance Tracking**: Automated history logging for the last 30 days.
* **Admin Notes**: Ability for administrators to leave specific notes/comments on employee records.
* **Responsive UI**: Clean, modern interface designed for both desktop and tablet use.

##  Installation & Setup

1.  **Clone the Repository**
    ```bash
    git clone [https://github.com/YOUR_USERNAME/AttendanceManagement.git](https://github.com/YOUR_USERNAME/AttendanceManagement.git)
    ```
2.  **Move to Web Directory**
    Place the folder in your `htdocs` (XAMPP) or `www` (WAMP) folder.
3.  **Database Setup**
    * Open `phpMyAdmin`.
    * Create a new database named `attendance_db`.
    * Import the provided `.sql` file.
4.  **Configuration**
    Update your database connection details in `config.php` or `db.php`.
5.  **Run**
    Navigate to `localhost/AttendanceManagement/login.php` in your browser.
